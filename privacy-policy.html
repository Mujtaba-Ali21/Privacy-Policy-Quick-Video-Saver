Privacy Policy

Effective Date: March 2026

This application respects your privacy and is committed to protecting user data. This app is designed to allow users to view and save media files that already exist on their device.

Information Collection

This application does not collect, store, transmit, or share any personal information from users. No analytics, tracking systems, or advertising identifiers are used.

File Access

The application may allow users to select a folder on their device using the Android system file picker. This is used only to display media files already stored on the device.

The app does not automatically scan device storage and does not access files without the user's explicit action.

Local Processing

All processing occurs locally on the user's device. No files are uploaded to external servers. Media files remain entirely under the user's control.

Third-Party Services

This application does not use third-party services that collect user data.

Data Security

Since the app does not collect or store user data, there is no personal information stored by the application.

Independence Disclaimer

This application is an independent utility tool designed to help users view media stored on their own device. It is not affiliated with, sponsored by, or endorsed by any messaging platform or third-party service.

Changes to this Policy
